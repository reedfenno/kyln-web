Project KYLN

Kinetic Yield Learning Network

An autonomous, multi-fidelity AI engine designed to discover novel actinide chelators at chemical accuracy. Bridging the gap between organic data abundance and relativistic f-block scarcity.

Forge

The Generator

Reactor

The Orchestrator

Manages the active learning lifecycle. Uses uncertainty sampling to select high-value candidates for DFT verification (ORCA) and autonomous retraining.

The Roadmap

From Foundation to Discovery

COMPLETED

Phase 1: Foundation

Pre-training on 9M organic conformations (ANI-2x). The model learns general chemical principles, forces, and invariance.

IN PROGRESS

Phase 2: Transfer

Transfer surgery and fine-tuning on 86k transition metal complexes (tmQM). Adapting the latent space from organic to coordination chemistry.

FUNDING REQUIRED

Phase 3: Active Learning

The autonomous discovery loop for Actinides (U, Np, Pu). This requires continuous retraining of the 5-member ensemble to drive the uncertainty sampling.

Bottleneck: Phase 3 requires a dedicated GPU (RTX 4090) to run the Kilonova trainer in loop with the DFT backend.
